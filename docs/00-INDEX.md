## 🔄 Conventions de Nommage

- **API** endpoints: HTTP verbs (GET, POST, PUT, DELETE)
- **Controllers**: [Resource]Controller (e.g., UserController)
- **Services**: I[Service]Service interface
- **Repositories**: I[Entity]Repository interface
- **DTOs**: [Entity][Direction]Dto (e.g., UserCreateDto)
